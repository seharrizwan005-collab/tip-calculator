Tip & Bill Splitter
A tip calculator and bill splitter that updates live as you type.
How to Run

No installs needed
Just open index.html in any browser
Or run npx serve . and open http://localhost:3000

What it does

Live results as you type — no calculate button
Tip presets: 10%, 15%, 20% + custom % input
Shows total tip, grand total, and per-person amount
Inline error messages near each field
Bill capped at Rs 1,000,000
Reset button (greys out when form is already empty)
Page title updates to show per-person amount live
Per-person amount always rounded up to nearest paisa
